Secure Network Design – 3 Floor Office

This project demonstrates my ability to design a secure, scalable, enterprise-style network for a multi-floor office building. The design includes VLAN segmentation, a Layer-3 core router for inter-VLAN routing, secure firewall policies, VPN remote access, and dual-SSID Wi-Fi for corporate and guest users.

Key Technical Highlights

  • Architecture: Internet → Firewall → Core Router → Access Layer
  • Inter-VLAN routing performed by Layer-3 Core Router
  • Department segmentation using VLANs:
    • VLAN 10 – HR → 10.10.10.0/24
    • VLAN 20 – Finance → 10.10.20.0/24
    • VLAN 30 – IT → 10.10.30.0/24
    • VLAN 40 – Servers → 10.10.40.0/24
    • VLAN 50 – Guest → 10.10.50.0/24
    • VLAN 60 – Management → 10.10.60.0/24
  • Servers grouped in VLAN 40 (File, App, Log/SIEM)
  • Firewall enforces east-west and north-south traffic policies
  • Remote users connect securely through VPN
  • Enterprise Wi-Fi design:
    • CorpNet → VLAN 30 → 10.10.30.0/24
    • GuestNet → VLAN 50 → 10.10.50.0/24
  • Diagram created in Draw.io using Cisco-style symbols

Presentation Script (Click to Expand)

For this project, I approached it as an IT consulting company designing a secure network for a three-floor office building. The goal was to create a segmented network that protects sensitive departments like HR and Finance while supporting IT operations, servers, and guest Wi-Fi.

Traffic flows from the Internet through a firewall to a Layer-3 core router, which performs inter-VLAN routing between HR, Finance, IT, Servers, Guest, and Management networks. Guest Wi-Fi is fully isolated, while CorpNet provides secure internal access. Remote employees connect through VPN for safe access to internal resources.

This design reflects real enterprise networking practices and demonstrates my understanding of VLANs, routing, firewall rules, wireless architecture, and secure network segmentation.
