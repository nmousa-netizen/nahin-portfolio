Secure Network Design – 3 Floor Office

This project demonstrates my ability to design a secure, scalable, enterprise-grade network for a multi-floor office building. The solution includes VLAN segmentation, a Layer-3 core router for inter-VLAN routing, firewall security, VPN remote access, and dual SSID Wi-Fi for corporate and guest users.

Key Technical Highlights

  • Architecture: Internet → Firewall → Core Router → Access Layer
  • Inter-VLAN routing performed by a Layer-3 Core Router
  • VLAN Segmentation Across Departments
    • VLAN 10 – HR → 10.10.10.0/24
    • VLAN 20 – Finance → 10.10.20.0/24
    • VLAN 30 – IT → 10.10.30.0/24
    • VLAN 40 – Servers → 10.10.40.0/24
    • VLAN 50 – Guest → 10.10.50.0/24
    • VLAN 60 – Management → 10.10.60.0/24
  • Servers placed in VLAN 40, including File Server, App Server, and Log/SIEM Server
  • Firewall applies both north-south and east-west security policies
  • Remote employees connect securely using a VPN server
  • Enterprise Wi-Fi Architecture
    • CorpNet → VLAN 30 → 10.10.30.0/24
    • GuestNet → VLAN 50 → 10.10.50.0/24
  • Network diagram created using Draw.io with Cisco-style symbols

Presentation Script (Click to Expand)

For this project, I approached it as an IT consulting company designing a secure and scalable network for a three-floor office building. The goal was to create a segmented environment that protects sensitive departments such as HR and Finance, while supporting IT operations, servers, remote access, and guest Wi-Fi.

Traffic flows from the Internet through a firewall into a Layer-3 core router, which performs inter-VLAN routing between HR, Finance, IT, Servers, Guest, and Management networks. The wireless design separates corporate and guest traffic, ensuring strong security and performance.

This project demonstrates my knowledge of VLANs, routing, security policies, firewall configuration, Wi-Fi architecture, and enterprise-level network design.
