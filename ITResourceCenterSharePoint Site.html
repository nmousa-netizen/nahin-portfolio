IT Resource Center – SharePoint Site (Michels)

I designed and built a modern SharePoint communication site that acts as a central IT Resource Center for Michels employees. The site brings together announcements, IT help, self-service tools, security training, and document governance in one easy-to-use place.

Key Features I Implemented

  • Branded landing page using Michels colors (red / white / black) with a welcome banner and clear messaging: “Everything you need for work resources, files, and announcements – all right here.”
  • Announcements section so IT can quickly share news, outages, and updates with all employees.
  • Quick links & navigation to:
    • IT help & support
    • Password reset and access requests
    • Security policies and guidelines
    • New hire / onboarding resources
    • Training & self-service how-to guides
  • Document libraries organized by category (policies, procedures, guides, templates) with metadata and views to make content easy to find.
  • Document life-cycle automation using Power Automate:
    • When a document is updated, users get email + Teams notifications.
    • Document owners receive alerts when others modify their content.
    • Every 3 months, owners get an automatic reminder to review/retire documents.
  • Self-service forms (SharePoint/Microsoft Forms) for reporting IT issues, submitting suggestions, and requesting access – all feeding into SharePoint lists for tracking.
  • Security & training area with links to security awareness training, phishing guidance, and best-practice checklists.
  • Analytics using SharePoint usage data and Power BI concepts to track which pages and documents are used most.

In this project I designed an IT Resource Center for Michels on SharePoint. The goal was to give employees one place for IT announcements, help, and documentation. I created a branded landing page with clear navigation, then built libraries for policies, procedures, and guides so content is organized and searchable.

I also automated the document life cycle using Power Automate: users are notified when content changes, and owners receive quarterly reminders to review their documents so information stays accurate. Finally, I added self-service forms and a security training area so employees can solve common IT problems on their own. This project shows my skills in SharePoint design, automation, and creating real business value with Microsoft 365.
