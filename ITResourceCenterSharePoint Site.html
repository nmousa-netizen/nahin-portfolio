IT Resource Center – SharePoint Site (Michels)

I designed and built a modern SharePoint communication site that serves as a central IT Resource Center for Michels employees. It brings together announcements, self-service IT tools, security training, and automated document governance in one user-friendly portal.

My Role in This Project

I served as the Project Coordinator, Business Analyst, and UX/UI Lead for this IT Resource Center initiative. I worked with stakeholders, gathered requirements, prioritized features, and ensured the final solution aligned with both technical and business needs.

  • Project Coordinator – managed timelines, organized tasks, facilitated team communication, and ensured project delivery.
  • Business Analyst – documented requirements, analyzed pain points, and translated business needs into SharePoint features.
  • UX/UI Lead – designed page layouts, navigation structure, user journeys, and visual elements.
  • Tools & Platforms Learned and Used:
    • Jira for project management and sprint planning
    • Figma for UX/UI wireframes and design mockups
    • Microsoft 365 + SharePoint for site creation and configuration

Key Features I Implemented

  • Branded landing page using Michels colors (red / white / black) with a modern welcome banner.
  • Built an AI Agent named MICO to enhance the user experience:
    • MICO searches the entire SharePoint site and document libraries
    • Finds documents by title and keywords
    • Reads inside documents (Word, PDF, Excel) to return accurate matches
    • Provides instant AI-generated summaries of any document
    • Helps employees discover policies, guides, and resources faster
  • User Analytics & Web Performance Dashboard built in Power BI:
    • Tracks what pages employees visit the most
    • Shows search trends and what users are looking for
    • Monitors document engagement (views, updates, downloads)
    • Highlights top IT resources accessed by the organization
    • Provides managers insights into website performance and employee needs
  • Announcements area for IT updates, alerts, and organization-wide communication.
  • Quick links to common IT needs:
    • Password reset & access requests
    • IT help & support
    • Security policies & best practices
    • Training & how-to guides
    • Onboarding resources
  • Organized document libraries with metadata, views, and structure for policies, procedures, and templates.
  • Document life-cycle automation using Power Automate:
    • Email + Teams alert when a document is updated
    • Owners notified when content changes
    • Quarterly reminders to review or retire documents
  • Self-service forms for IT requests and suggestions (Forms → SharePoint Lists).
  • Security training hub including phishing guidance and compliance resources.
  • Analytics insights using SharePoint usage and Power BI concepts.

In this project, I built a modern IT Resource Center for Michels using SharePoint. My goal was to centralize IT communication, documentation, and self-service tools so employees could easily find what they need.

I served as Project Coordinator, Business Analyst, and UX/UI Lead – managing timelines, gathering requirements, and designing the overall user experience. I created a branded landing page, organized document libraries, automated document governance with Power Automate, and added self-service forms for IT support and security processes.

I also built an AI Agent named MICO, which can search the entire site, find documents by title or keyword, read inside files, and return fast AI-generated summaries. On top of that, I designed a User Analytics & Web Performance Dashboard in Power BI so managers can see which pages and resources employees use the most and what they are searching for.

This project demonstrates my ability to design enterprise portals, integrate AI, use analytics, streamline IT workflows, and deliver real business value using Microsoft 365.
