Security Awareness Dashboard

SharePoint • Power Automate • Power BI

Overview

I built an integrated Security Awareness Center inside the SharePoint IT Resource site. Employees can quickly report suspicious emails, access security training, read policies, and request access. A Power Automate flow logs phishing reports into a SharePoint list and sends Teams alerts to IT. Then, using Power BI, I created a live dashboard showing:

  • Number of phishing emails reported
  • Suspicious files detected
  • Severity levels
  • Security event trends

Key Features

  • SharePoint Security Quick Links (Training, Policies, Password Reset, Report Suspicious Email)
  • Power Automate workflow for instant phishing alerts
  • SecurityEvents SharePoint List for logging incidents
  • Power BI dashboard for real-time visibility
  • End-user friendly interface with fast navigation
